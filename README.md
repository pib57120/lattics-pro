## Lattics 到底是什么

简单说：它是一款让你用**卡片搭积木写文章**的工具，适合 Win 和 Mac 双平台。

市面上的笔记软件，要么是 Notion 那种强调数据库的"项目管理风"，要么是 Obsidian 那种靠插件堆砌的"极客风"。Lattics 不走寻常路，它的思路更接近人脑的运转方式：先散点记录，再串联成网，最后梳理成文章。

用少数派某位作者的话来说，上手使用手册甚至比 flomo 还简洁，花个十几分钟就能摸透基本逻辑。如果你被 Obsidian 动辄上百页文档劝退过，Lattics 可能是个解压版选项。

## 几个让人眼前一亮的功能

**多维内容图谱**：切换视图就能看到文章和卡片之间的关系网络，拖拽节点整合知识，有点像把思维导图和文档编辑器捏在一起。

**双向链接引用**：输入 `@` 就能把其他项目引用进来，一处修改多处联动。写长文、做研究特别好用，改了一个定义，引用了这个定义的段落都会跟着更新。

**卡片式写作**：用 `/` 插入内容块，写着写着可以随时把一段话转成独立卡片，往后可以复用、拼装、拖拽进其他文章——有点"搭积木"的感觉。

**专业学术功能**：支持 Markdown、数学公式、论文模板、Zotero 链接、图文分栏，这套组合对在读生和研究者来说已经很齐全了。

**打字机模式 + 双屏对照**：写作的时候开打字机模式，视线始终跟着当前行走；双屏对照适合对照参考资料写东西，体验挺顺滑的。

**数据离线优先**：数据存本地，全库可备份，不用担心哪天服务器关了数据没了。支持 DOCX / PDF / MD 多格式导出，也可以打包共享。

## 基础版 vs Pro 版，差在哪

免费版已经能用了，无限文档、无限卡片，双向链接、多栏排版这些核心功能都有。但有一个硬限制：**知识图谱节点只有 15 个**，一旦超出，图谱和鸟瞰图功能直接灰掉，不能继续用。

对于只是轻度试用的人，免费版够了。但如果你打算正儿八经用来整理知识、写论文或者长文写作，15 个节点很快就不够用，而且导出 PDF / DOCX 也是 Pro 版专属功能。
